## 🌟 المميزات الرئيسية
- 🔍 مسح الشبكة باستخدام `arp-scan` و `nmap`
- 📊 توليد تقارير تلقائية في `~/GT-IpNet_Reports`
- 🌐 دعم متعدد اللغات (العربية/الإنجليزية)
- 📱 حفظ سجل الأجهزة في `~/GT-IpNet_Devices`
